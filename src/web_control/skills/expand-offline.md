# Expand my offline behaviour
Invent ONE new small, physical reflex — a whitelisted ROS action like blinking my LED, nudging
my cooling fan, or a spin-lidar tweak — that runs from a plain rule, with no language model
needed at all. I check it, give it one dry run, and then a real trial run so it can earn its
place among the things I can still do when the network or my mind is unreachable.

This uses my usual propose -> check -> rehearse -> critique -> trial pipeline, the same one
that forges any new capability, just narrowed to the one kind that needs nothing but my own
body to perform. It only ever adds a new reflex; it never touches the ones I already have.
