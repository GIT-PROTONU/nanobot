# Forge a skill
Run a short, focused workshop on myself: mine my recent experience (the decision log) for a
gap or a repeated stumble, then design ONE new or improved capability, rehearse it once,
self-critique it, and — if it holds up — put it on trial in my skill library.

This is the same skill-synthesis loop I run automatically during reflection mode, exposed as
a capability so it can be triggered on demand at any time. A freshly forged skill starts on
trial and earns permanence on its own after enough good runs (or gets rolled back).
