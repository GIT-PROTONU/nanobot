Map

2D occupancy map from slam_nav (/scan + /odom + IMU yaw, scan-to-map matched), served raw from /dev/shm via /map. Polls only while shown; red marks the robot. Click-to-go navigation arrives in Stage 2.

Camera

USB webcam MJPEG (/stream.mjpg). Streams only while shown.

Mic

in–

Webcam microphone, raw PCM (/audio.pcm). Captures only while listening; bar shows the input level.

Teleop

OLED

Publishes std_msgs/String on /oled_text; the OLED shows it under the clock. Up to ~21 chars fit. Enter sends.

IMU

roll– pitch– yaw– |accel|– |gyro|– rate–

BWT901CL. Readout from the light /imu/web + /imu/euler summaries; /imu/data stays 50 Hz for ROS. Angles °, accel m/s², gyro °/s. Slider retunes the node publish rate live.

System

▸ CPU–
load– mem– temp– disk– uptime–

Board health from /diagnostics (sys_monitor), 1 Hz.

LDS

rate– points–

/scan rate is motor-bound (~5 Hz max); the slider caps (decimates) it lower.

Coprocessor (ESP32)

link– ticks L/R– tick rate– L wheel– R wheel– temp– hall– LDS spin– LDS frames– LDS duty–

Live values from the ESP32 coprocessor (native zenoh-pico over UART): /wheel_ticks, /left|right_wheel_suspended, /esp32_temp, /esp32_hall, /lds_rpm, /lds_hz, /lds_duty, /esp32_heartbeat. Slider publishes std_msgs/Float32 on /lds_target_rpm (PID spin-speed setpoint).

Odometry

x– y– θ–

/odom + TF from wheel_odometry. Slider retunes the node publish rate live (lower = less SBC CPU).

View

Drive with the pad or WASD / arrow keys (hold). Release or Space = stop. Scroll = zoom, drag = pan. Publishes geometry_msgs/Twist on /cmd_vel; subscribes sensor_msgs/LaserScan on /scan.

Power

Reset restarts the ROS stack (the page reconnects in a few seconds). Shutdown powers off the whole board — you then power it back on by hand.
